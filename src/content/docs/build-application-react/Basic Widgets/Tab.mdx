> **Tab** is a widget that provides a tab widget that you can use in your application.

## Example

export const tabExample = `
import { Tab, TabItem } from '@vixen-js/core-react';

export function TabExample() {
    return (
        <Tab>
            <TabItem text="Tab 1" />
            <TabItem text="Tab 2" />
            <TabItem text="Tab 3" />
        </Tab>
    );
}
`;
