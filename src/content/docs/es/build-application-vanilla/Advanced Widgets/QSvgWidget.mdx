> **QSvgWidget** es un widget que proporciona un SVG que puedes utilizar en tu aplicación.

## Ejemplo

export const qSvgWidgetExample = `
import fs from "fs";
import { QSvgWidget } from '@vixen-js/core';

const svg = new QSvgWidget();
svg.load("./assets/image.svg");
// o
fs.readFile("./assets/image.svg", (err, buff) => {
    if (err) throw err;
    svg.load(buff);
});
`;
