> **Slider** is a widget that provides a slider that you can use in your application.

## Example

export const sliderExample = `
import { Slider } from '@vixen-js/core-react';

export function SliderExample() {
    return (
        <Slider 
          value={50} 
          minimum={0} 
          maximum={100} 
          tickInterval={2} />
    )
}
`;
