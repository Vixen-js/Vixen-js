### Automatic Installation

We recommend starting new Vixen UI project using [`@vixen-js/create-vixen`](https://github.com/Vixen-js/create-vixen), which setups all things you need automatically. To create a project, run:

<Code code={outputInstall} lang='bash' />
On Installation process, you'll see the following output:

<Code code={outputCreate} lang='bash' />

After the output, `@vixen-js/create-vixen` will create a folder with your project name with all the dependencies required to start working with Vixen UI.

if you're new to Vixen UI, check out the [Tutorials](/build-application) section.
