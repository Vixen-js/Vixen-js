> **QStackedWidget** Provee un widget que apila varios widgets donde solo un widget es visible a la vez.

## Ejemplo

export const qStackedWidgetExample = `
import { QStackedWidget, QWidget, QLabel, QMainWindow } from '@vixen-js/core';

const win = new QMainWindow();

const stack = new QStackedWidget();

const first = new QLabel();
first.setText("First");

const  second = new QLabel();
second.setText("Second");

stack.addWidget(first);
stack.addWidget(second);

win.setCentralWidget(stack);
win.resize(300, 300);
win.show();
`;
