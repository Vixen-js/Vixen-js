> **QBoxLayout** is a layout manager that organizes widgets in a __horizontal__ or __vertical__ layout.

## Example

export const qBoxLayoutExample = `
import { QBoxLayout, QCalendarWidget, QWidget} from '@vixen-js/core';

const widget = new QWidget();
const box = new QBoxLayout(0);

box.addWidget(new QCalendarWidget());
box.addWidget(new QCalendarWidget());

widget.setLayout(box);
`;
