> **QWidget** es el widget base de todo widget de Qt.

Puedes usar QWidget para crear tus propios widgets.

## Ejemplo

export const qWidgetExample = `
import { QWidget, FlexLayout } from '@vixen-js/core';

const widget = new QWidget();
widget.setObjectName("container");
widget.setLayout(new FlexLayout());
`;
