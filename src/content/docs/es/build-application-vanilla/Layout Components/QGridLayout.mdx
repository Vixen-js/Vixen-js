> **QGridLayout** is a layout manager that organizes widgets in a __grid__ layout.

## Example

export const qGridLayoutExample = `
import { QGridLayout, QWidget, QLabel } from '@vixen-js/core';

const view = new QWidget();
const grid = new QGridLayout();

view.setLayout(grid);

const title = new QLabel();
title.setText("Hello World");

const subtitle = new QLabel();
subtitle.setText("This is a subtitle");

grid.addWidget(title);
grid.addWidget(subtitle);
`;
