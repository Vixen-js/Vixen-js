> **QButtonGroup** provee un contenedor abstracto en el que se pueden agregar botones de widgets. no provee ninguna representación visual de este contenedor.

## Ejemplo

export const qButtonGroupExample = `
import { QButtonGroup, QWidget, QLabel, QButton } from '@vixen-js/core';

const widget = new QWidget();
const group = new QButtonGroup();

const button1 = new QButton();
button1.setText("Button 1");
button1.setObjectName("button1");
group.addButton(button1);

const button2 = new QButton();
button2.setText("Button 2");
button2.setObjectName("button2");
group.addButton(button2);

widget.setLayout(group);
`;
