> **View** es un widget que proporciona un componente View que puedes usar en tu appellate.

Puedes usar este componente para crear un contenedor parecido a un `<div>` en HTL en el mundo web.

## Ejemplo

export const viewExample = `
import { View, Text } from '@vixen-js/core-react';

export function ViewExample() {
    return (
        <View>
            <Text>Hello World</Text>
        </View>
    )
}
`;
