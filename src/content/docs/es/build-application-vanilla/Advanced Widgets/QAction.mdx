> **QAction** es un widget que proporciona una acción que puedes utilizar en tus menus.

## Ejemplo

export const qActionExample = `
import { QAction, QMenu } from '@vixen-js/core';

const menu = new QMenu();
const action = new QAction();

action.setText("Your action");
action.addEventListener("onTrigger", () => {
    console.log("Action triggered");
});
menu.addMenu(action);
`;
