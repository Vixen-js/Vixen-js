> **BoxView** es un layout manager que organiza widgets en un layout __horizontal__ o __vertical__.

## Ejemplo

export const boxLayoutExample = `
import { BoxView, Text } from '@vixen-js/core-react';
import { Direction } from '@vixen-js/core';

export function BoxLayoutExample() {
    return (
        <BoxView direction={Direction.TopToBottom}>
            <Text>Hello World</Text>
        </BoxView>
    )
}
`;
