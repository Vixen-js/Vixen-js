> Vixen UI use `Esbuild` as its build tool, so if you are familiar with Esbuild, you can see some similar project structure.

The project is organized into the following `top-level` directories and files:

<FileTree>
- src 
    - asssets
        - images
            - AppIcon.png Required to build the application, the image should have 1024x1024 pixels in size
            - ... Place here all the image resources
        - styles
            - ... Place here all the CSS style resources
    - main.ts Application Entrypoint
    - vixen-env.d.ts
- eslint.config.js
- package.json
- tsconfig.json
- build.mjs Esbuild build script
</FileTree>
