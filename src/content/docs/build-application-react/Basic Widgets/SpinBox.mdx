> **SpinBox** is a widget that provides a spin box that you can use in your application.

## Example

export const spinBoxExample = `
import { SpinBox } from '@vixen-js/core-react';

export function SpinBoxExample() {
    return (
        <SpinBox range={{ minimum: 0, maximum: 100}} value={20} singleStep={1} />
    );
}
`;
