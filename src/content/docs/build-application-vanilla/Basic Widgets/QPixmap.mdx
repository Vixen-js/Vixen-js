> **QPixmap** is a widget that provides an image that you can use in your application.

## Example

export const qPixmapExample = `
import { QPixmap, QLabel } from '@vixen-js/core';

const imageUrl = "../../assets/images/hello-world.png";
const pix = new QPixmap(imageUrl);
const image = new QLabel();
image.setPixmap(pix);
`;
