> **Image** es un componente personalizado que envuelve una QLabel y QPixmap para mostrar una imagen en tu aplicación.

## Ejemplo

export const ImageExample = `
import { Image } from '@vixen-js/core-react';

export function ImageExample() {
    return (
        <Image
            src="./assets/images/hello-world.png"
        />
    );
}
`;
