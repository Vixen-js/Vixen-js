> **QProgressDialog** es un widget que proporciona un QProgressDialog que puedes utilizar en tu aplicación.

## Ejemplo

export const qProgressDialogExample = `
import { QProgressDialog } from '@vixen-js/core';

const progress = new QProgressDialog();
progress.setLabelText("Downloading...");
progress.setCancelButtonText("Cancel");
progress.setRange(0, 100);
progress.setValue(0);
`;
